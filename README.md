## 📖 English Section

### Overview
CyberSec-Prompts-Hub is a comprehensive, bilingual (Arabic/English) collection of professionally crafted AI prompts designed for cybersecurity professionals, penetration testers, security researchers, and SOC teams. Our prompts follow industry standards including NIST, OWASP, and MITRE ATT&CK frameworks.

### ✨ Features
- **500+ Professional Prompts** covering all cybersecurity domains
- **Bilingual Support** - Full Arabic and English versions for each prompt
- **Industry Standards** - Aligned with NIST, OWASP, MITRE ATT&CK
- **Real-World Examples** - Actual tool outputs and scenarios
- **Multi-Model Support** - Optimized for GPT-4, Claude, Gemini, and local LLMs
- **Organized Structure** - Easy-to-navigate category hierarchy


2. Browse prompts by category in the `prompts/` directory

3. Use with your favorite AI model (ChatGPT, Claude, Gemini, etc.)

## 📖 القسم العربي

### نظرة عامة
CyberSec-Prompts-Hub هي مكتبة شاملة وثنائية اللغة (عربي/إنجليزي) من مطالب الذكاء الاصطناعي المصممة بشكل احترافي لمتخصصي الأمن السيبراني، مختبري الاختراق، الباحثين الأمنيين، وفرق مركز عمليات الأمن (SOC). تتبع مطالبنا معايير الصناعة بما في ذلك أطر NIST و OWASP و MITRE ATT&CK.

### ✨ المميزات
- **أكثر من 500 مطلب احترافي** يغطي جميع مجالات الأمن السيبراني
- **دعم ثنائي اللغة** - نسخ عربية وإنجليزية كاملة لكل مطلب
- **معايير الصناعة** - متوافق مع NIST و OWASP و MITRE ATT&CK
- **أمثلة واقعية** - مخرجات أدوات فعلية وسيناريوهات حقيقية
- **دعم نماذج متعددة** - محسّن لـ GPT-4 و Claude و Gemini والنماذج المحلية
- **هيكل منظم** - تسلسل هرمي للفئات سهل التصفح

### 🚀 البدء السريع
1. استنساخ المستودع:
```bash
git clone https://github.com/Rootx202/CyberSec-Prompts-Hub.git
```

2. تصفح المطالب حسب الفئة في مجلد `prompts/`

3. الاستخدام مع نموذج الذكاء الاصطناعي المفضل (ChatGPT، Claude، Gemini، إلخ)
