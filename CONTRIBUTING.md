## 🇬🇧 English

### How to Contribute

We welcome contributions from the cybersecurity community! Here's how you can help improve CyberSec-Prompts-Hub.

#### Ways to Contribute

1. **Add New Prompts** - Submit new prompts following our template
2. **Improve Existing Prompts** - Enhance clarity, add examples, or fix errors
3. **Translate Prompts** - Help improve Arabic translations
4. **Fix Typos** - Correct any spelling or grammar mistakes
5. **Suggest Categories** - Propose new categories or subcategories

#### Contribution Process

1. **Fork the Repository**
   ```bash
   git clone https://github.com/Rootx202/CyberSec-Prompts-Hub.git
   ```

2. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Your Changes**
   - Follow the prompt template below
   - Ensure both Arabic and English versions are included
   - Test your prompt with at least one AI model

4. **Commit Your Changes**
   ```bash
   git commit -m "Add: Brief description of your changes"
   ```

5. **Push and Create Pull Request**
   ```bash
   git push origin feature/your-feature-name
   ```

#### Quality Guidelines

- ✅ Prompts should be clear and specific
- ✅ Include practical examples
- ✅ Use professional cybersecurity terminology
- ✅ Follow NIST/OWASP standards where applicable
- ✅ Arabic translations should use formal technical Arabic (فصحى تقنية)
- ❌ No malicious or illegal content
- ❌ No promoting harm or unauthorized access

## 🇸🇦 العربية

### كيفية المساهمة

نرحب بمساهمات مجتمع الأمن السيبراني! إليك كيف يمكنك المساعدة في تحسين CyberSec-Prompts-Hub.

#### طرق المساهمة

1. **إضافة مطالب جديدة** - تقديم مطالب جديدة تتبع قالبنا
2. **تحسين المطالب الموجودة** - تعزيز الوضوح، إضافة أمثلة، أو إصلاح الأخطاء
3. **ترجمة المطالب** - المساعدة في تحسين الترجمات العربية
4. **إصلاح الأخطاء الإملائية** - تصحيح أي أخطاء إملائية أو نحوية
5. **اقتراح فئات** - اقتراح فئات أو فئات فرعية جديدة

#### عملية المساهمة

1. **تفريع المستودع**
   ```bash
   git clone https://github.com/Rootx202/CyberSec-Prompts-Hub.git
   ```

2. **إنشاء فرع**
   ```bash
   git checkout -b feature/اسم-الميزة-الخاص-بك
   ```

3. **إجراء التغييرات**
   - اتبع قالب المطلب أدناه
   - تأكد من تضمين النسخ العربية والإنجليزية
   - اختبر مطلبك مع نموذج ذكاء اصطناعي واحد على الأقل

4. **تثبيت التغييرات**
   ```bash
   git commit -m "إضافة: وصف مختصر لتغييراتك"
   ```

5. **الدفع وإنشاء طلب السحب**
   ```bash
   git push origin feature/اسم-الميزة-الخاص-بك
   ```

#### إرشادات الجودة

- ✅ يجب أن تكون المطالب واضحة ومحددة
- ✅ تضمين أمثلة عملية
- ✅ استخدام مصطلحات الأمن السيبراني المهنية
- ✅ اتباع معايير NIST/OWASP حيثما ينطبق
- ✅ يجب أن تستخدم الترجمات العربية العربية الفصحى التقنية
- ❌ لا محتوى ضار أو غير قانوني
- ❌ لا ترويج للأذى أو الوصول غير المصرح به
