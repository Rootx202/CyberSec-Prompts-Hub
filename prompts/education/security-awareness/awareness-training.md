## Prompt 1: Phishing Awareness Training

### Description
Develop comprehensive phishing awareness training content.

### Tags
`phishing` `security-awareness` `training` `social-engineering` `education`

---

## 🇬🇧 English Prompt

```
Develop phishing awareness training content for employees:

Target Audience: [PROVIDE AUDIENCE DETAILS]

Create training covering:

1. Phishing Fundamentals
   - What is phishing?
   - Types of phishing attacks
   - Why employees are targeted
   - Statistics and impact

2. Identifying Phishing Emails
   - Suspicious sender addresses
   - Urgent language and threats
   - Mismatched URLs
   - Grammar and spelling errors
   - Unexpected attachments
   - Requests for sensitive information

3. Phishing Examples
   - Real-world examples
   - Industry-specific examples
   - Current campaign trends
   - Simulated phishing samples

4. Response Procedures
   - How to report suspected phishing
   - What to do if clicked
   - Who to contact
   - Incident reporting process

5. Prevention Best Practices
   - Email security hygiene
   - Verifying requests
   - Safe browsing practices
   - Multi-factor authentication

Provide training content, quiz questions, and handout materials.
```

┌─────────────────────────────────────────────────────┐

│ 3. Malware/ransomware                  18%         │
│ 4. Misconfiguration                    12%         │
│ 5. Physical theft                       9%         │
└─────────────────────────────────────────────────────┘

Prevention Measures:
✓ Encrypt sensitive data at rest and in transit
✓ Use strong access controls (least privilege)
✓ Implement DLP (Data Loss Prevention)
✓ Regular security training
✓ Monitor and audit data access
✓ Have incident response plan ready

MODULE 5: CLOUD AND MOBILE DATA
─────────────────────────────────────────────────────

✓ Company SharePoint/OneDrive
✓ Approved cloud services only

- Install security updates promptly
